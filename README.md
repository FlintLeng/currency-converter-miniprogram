# 汇率转换微信小程序

一个简洁美观的实时汇率转换工具，支持多种货币之间的快速转换。

## 功能特点

- ✅ 实时汇率获取
- ✅ 支持7种常用货币（人民币、美元、港币、欧元、日元、新加坡元、韩元）
- ✅ 一键交换货币
- ✅ 点击复制结果
- ✅ 显示其他货币兑换结果
- ✅ 美观的UI设计

## 使用说明

1. 下载微信开发者工具
2. 导入项目文件夹
3. 在微信开发者工具中预览和调试
4. 点击"上传"发布小程序

## 汇率API

项目使用 `exchangerate-api.com` 免费API获取实时汇率。

如需使用其他API，可修改 `utils/exchangeApi.js` 文件。

## 注意事项

- 实际发布时需要在微信公众平台注册小程序账号
- 汇率API有请求频率限制，建议添加缓存机制
- 生产环境建议使用付费API获取更稳定的汇率数据
